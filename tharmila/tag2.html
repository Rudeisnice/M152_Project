Fonts
  • Mit Hilfe der CSS-Regel @font-face, lassen sich auf einfache Weise Webfonts in eine Webseite einbinden. Die Webfonts befinden sich in der Regel gemeinsam mit den Webseiten auf einem Server. Sie werden vom Browser bei Bedarf heruntergeladen und zur Darstellung verwendet.
  • Seit 2011 unterstützen die modernen Browser »Woff« einhellig als Schrift-Format. Nur für IE8 muss weiterhin EOT angeboten werden.
Schriftfornamte
  • TrueType Fonts (TTF) → Meistverwendetes Font Format für Mac und Windows
  • The Web Open Font Format (WOFF) → Gebrauch in web pages. TrueType/OpenType Schrift. W3C Empfehlung.
  • The Web Open Font Format (WOFF 2.0) → TrueType/OpenType font welche eine bessere Kompression erlaubt statt WOFF 1.0.
  • Embedded OpenType Fonts (EOT) → EOT fonts sind eine kompakte Form von OpenType fonts designed by Microsoft. Supported nur vom IE.
Social Media Icons
  • Icons in eine Webseite einzufügen ist nicht schwer. Aber wenn man Texte und Icons kombiniert (z.B. für einen Button) oder wenn die Icons in der Grösse veränderbar sein sollen, dann wird es schwierig.
  • Die Lösung für dieses Problem sind Icon Fonts. Icon Fonts sind Schriftarten, welche anstatt Buchstaben nur Icons enthalten. Dadurch können diese Icons genauso verwendet werden, wie gewöhnliche Schriften, d.h. man kann ihre Grösse und auch die Farbe verändern.
Icomoon einbinden
  • Ordner fonts und style.css benötigt → in eigenes Projekt einbinden mit @font face
  • Icons in Html einbinden mit class name in ein "span"
  • Css Anpassungen falls nötig
