Dokumentation 20.12.2017

Notizen

Auf fontsquirrel.com kann man gratis Schriften runterladen und ausserdem gibts da noch einen Font Generator, welcher in alle Formate umwandelt. Mit @font-face kann man leicht Webfonts in Webseiten einbinden. Diese befinden sich meist zusammen mit den Webseiten auf einem Server und können von da bei bedarf vom Browser runtergeladen und dann dargestellt werden. Die wichtigsten Schriftformate sind TrueType Fonts (oder TTF, wird am meisten verwendet), The Web Open Font Format (auch WOFF, wird in Webpages verwendet), (WOFF 2.0, das eine bessere Kompression besitzt als WOFF 1.0) und Embedded OpenType Fonts (EOT). Icon Fonts sind Schriftarten, die statt einfachen Buchstaben Icons enthalten. So kann man solche ohne Probleme in einen Text eingliedern und auch die Farbe wechseln, wie bei einer normalen Schrift. Auf icomoon.io kann man Icons aussuchen und downloaden. Mit <\span class="icon-x"><\span> wird so ein Icon dann eingefügt.
